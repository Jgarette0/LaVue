
  ##Steps run this project into your local (ithink)

  


1.
        
                git clone https://github.com/Jgarette0/LaVue.git

        
2.
        
                cd lavue
        
3.
        
              composer install
        
4.
        
                npm install
        
5.
        
                cp .env.example .env
        
6.
        
                php artisan key:generate

7.
        
                 npm run dev
        
8.
        
                 php artisan serve
